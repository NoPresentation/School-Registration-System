Welcome to the application!

Before running the app, make sure of the following:
1. Python (3.11.9)
2. All the needed libraries from 'req' file, which you can install in your terminal by using the command
'pip install -r req'. Note: Make sure you are in the application directory when running that command.

Executing the applicatoin:
1. Navigate to the application directory in your terminal
2. Run the command 'python main.py'
3. Enjoy!
